How To Use:
Step 1 — Set your working folder

In chat, type:

set folder C:\Users\YourName\Documents\TestFolder

(Use your actual folder path)

The assistant will now read and index your documents.

Step 2 — Ask questions (RAG Chatbot)

Examples:

When is my exam?
Explain chapter 2 from my notes
What is poultry vaccination?

Step 3 — Organize files safely

Type:

organize files

You will see a preview of how files will be grouped.

Then confirm:

confirm

Files will be moved into context-based folders.
